## ⬇️ Instalación en termux

[Termux](https://play.google.com/store/apps/details?id=com.termux&hl=pt_BR&gl=US)

```bash
> pkg update -y
> pkg upgrade -y
> pkg install git
> git clone https://github.com/g4tito/gati-bot
> cd gati-bot
> bash install.sh
> npm start
```

#### iniciar el bot manualmente

```bash
> npm start
```

#### reiniciar el bot automáticamente

```bash
> sh start.sh
```
