# ⬇️ Instalación

[Termux](https://play.google.com/store/apps/details?id=com.termux&hl=pt_BR&gl=US)

```
> git clone https://github.com/g4tito/gati-bot
> cd Brizas-bot
> bash install.sh
> npm install

(iniciar el bot manualmente)
> npm start

(reiniciar el bot automáticamente en caso de crasheo)
> sh start.sh

(o tambien)
> pm2 start index.js

```
